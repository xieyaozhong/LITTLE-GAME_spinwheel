# 露西霓虹鎖定更新包

這個檔案包移除了露西的可愛、愛心、撒嬌式設定，保留中性冷調的霓虹駭客風格。

## 包含檔案

- `lucy.js`
  - 露西像素角色模型
  - 粉紫、藍紫、薄荷綠髮色
  - 深色科技服裝與紅色訊號細節
  - 中性角色說明與台詞

- `lucy.css`
  - 角色待機動畫
  - 霓虹掃描環
  - 鎖定光束
  - 資料流、網格、輪框燈光
  - 中獎鎖定脈衝

- `behaviors.js`
  - 露西不使用一般轉盤旋轉
  - 逐格掃描所有獎項
  - 最後直接以 Canvas 霓虹覆蓋層鎖定中獎區塊
  - 保留其他角色原本的抽取邏輯

- `labels.js`
  - 在角色選單加入露西
  - 依序載入 `lucy.js` 與 `behaviors.js`

- `styles.css`
  - 載入 `lucy.css`

## 安裝方式

將這五個檔案覆蓋到專案根目錄：

```text
lucy.js
lucy.css
behaviors.js
labels.js
styles.css
```

重新部署 GitHub Pages 後，建議用無痕模式測試，以避免瀏覽器快取舊版 JavaScript 與 CSS。

## 露西抽取流程

1. 啟動神經連線
2. 逐格掃描輪盤獎項
3. 霓虹覆蓋層逐項移動
4. 偵測中獎區塊
5. 光束與脈衝鎖定
6. 顯示結果

輪盤本體不會進行一般的高速旋轉。
